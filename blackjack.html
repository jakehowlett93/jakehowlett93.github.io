Blackjack

This blackjack game is one of the first games i made whilst learning to code in Python. It was my first attempt at taking an object orientated approach and I gained a good understanding of how classes work and how they can be useful. It also gave me a chance to get lots of practice using functions and loops, having only used them in an abstract way up until this project, I enjoyed putting them to use in a project where I had to consider how the functions and loops would interact and zone in on exactly what i wanted them to do. My favourite part about this project is the game loop, it was challenging figuring out how all my functions and classes fitted together properly but it was very satisfying when I finally worked it out and fixed all the bugs, especially when I got to play the game and see all my hardwork come together.
