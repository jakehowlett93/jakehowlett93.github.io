Survey Page

I made this webpage as one of the projects on freecodecamp that is needed to get the Responsive Web Design Certification. The main focus of this project was to practice using form elements, I learned about the different elements and some of their features, for example how to make certain elements a requirement. I also wanted to practice using some of HTML5's features to improve accessibility, namely article, section, and header. Similarly I used labels and the name attribute to make the form screen reader friendly. My favourite part of this project is that I found a simple way to split the page into two columns using just HTML and CSS which made it easy to get a clean looking layout for the form.To view the code click here.
